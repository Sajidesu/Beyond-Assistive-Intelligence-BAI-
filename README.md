Backend(Server):

There must be two seperate virtual environments. One environment is for the main backend (program) and another backend for TaskServer and google_trial.

run program in port 8000 and Taskserver in 8001.

Front-end(UI):

Paste the files in Android Studio with the Kotlin JDK. 

Android Studio will ask to 'Sync' the files. Press Sync Now.

Then press the 'run' button above to run the application.

APP:

Download the app, run the server with the same local host of the device running the app.
